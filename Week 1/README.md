# Profile App – Flutter Flavors

This project extends the Profile App assignment by implementing Flutter App Flavors to support different application environments. 

## 📱 Flavors

- Development (dev)
- Staging (staging)
- Production (prod)

## ✅ Setup 

1. Configured app flavors using the flutter_flavorizr package

2. Created three flavors:
   - dev
   - staging
   - prod

3. Assigned application IDs and bundle IDs for each flavor with different named
4. Replaced user choosen launcher icons within
   - app/src/dev/res
   - app/src/prod/res
   - app/src/staging/res

## 🚀 Run Commands 

### Flavor App
flutter run --flavor dev

### Staging App
flutter run --flavor staging

### Production App
flutter run --flavor prod

## 📚 Learning Outcomes

- Flutter app flavors
- Android application IDs
- iOS bundle identifiers
- Different mobile applications environment deployment
